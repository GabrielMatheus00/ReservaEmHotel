Basicamente, criei um sistema para cadastros de hoteis e quartos associados aos mesmos, onde o valor de cada quarto é em dólar e ele utiliza um schedule job no hangfire para pegar a cotação do dia e calcular os valores em reais.

Arquiteturas utilizadas:
- DDD
- Microsservices
- Repositories
- Unit Of Work

Tecnologias Utilizadas
- .Net 8
- Postgresql
- Docker
- Hangfire
